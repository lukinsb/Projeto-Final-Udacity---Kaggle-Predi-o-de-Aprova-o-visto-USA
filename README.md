# Predição de Aprovação em Visto Permanente Americano

## Proposta

Projeto final para aprovação no nanodegree Engenheiro de Machine Learning da Udacity

### Prerequisitos

* Python => 3.6
* Pandas
* Numpy
* Cufflinks
* Plotly
* Seaborn
* matplotlib

### Instalação

Todos os pacotes estão disponíveis na distribuição anaconda ou por meio do gerenciador PiPy

## Testes

O notebook python deve ser mantido na mesma pasta em que o dataset, estando pronto para execução como distribuído.
Link para o Dataset (~240MB): https://www.kaggle.com/jboysen/us-perm-visas/downloads/us-perm-visas.zip/1
